#  开发约定

-----

## 字体

> * 字体、边框、圆角、阴影用px单位、除此以外都用rem [强制]
> * 字体大小在12px, 14px, 16px, 18px内选取
> * 12px 小号字体
> * 14px 普通字体
> * 16px 大号字体
> * 18px 特大字体
> * 选取字体时应尽量贴近设计稿, 参考公式:设计稿字体尺寸/2

-----

## 内容

> * 所有的内容需放在 #wrapper内 [强制]

-----

## 作用域

> * 为方便后期合并css、 所有的样式都需约束在页面作用域里(或模块作用域) [强制]
> * js 作用域约束在匿名自执行函数内

-----

## 其他

> * 静态资源引用需定位到public文件夹 [强制]
